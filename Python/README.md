## install_package_python

Carpeta que instala los paquetes de python enlistados en el archivo package_python3.csv

### Ejecución

```console
./install.sh
```

### Contenido

- **install_package.py**:

  Realiza la instalación de los paquete de Ubuntu con el uso de python3.

- **Paquetes a instalar**:

Flask

numpy
