## Tiling window manager - Pop shell

Carpeta que contiene el instalador del windows managaer [pop shell](https://github.com/pop-os/shell) y sus shortcuts.

### Ejecución

```console
./install.sh
```

### Contenido

- **install.sh**:

  Realiza la instalación de el windows manager pop-shell.
